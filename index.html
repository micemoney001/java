# JAVA JAVASCRIPT
1 It’s a class-based object-oriented language. It’s a prototype-based scripting language.
2 It’s a stand-alone programming language that is run on the Java Virtual by Machine. Here the source code is converted into byte code. JavaScript is a scripting language that must be placed within an HTML document. It should be run on various web browsers such as Firefox, chrome and others.
3 The file extension of Java is . Java The file extension of Javascript is .js
4 It’s compiled and interpreted where source code is translated into bytecodes by Java Virtual Machine. There’s a JavaScript interpreter in each browser for the execution of JavaScript code
5 The code type of Java is Object-Oriented The code type of Javascript is Object-Based
6 Data types have to be declared Data types don’t have to be declared.
7 Compiled on the server prior to execution on the client Interpreted by client
8 It’s a class-based object-oriented language. It’s a prototype-based scripted language
9 It supports static typing and that requires type declaration and variable prior to compilation and execution. It supports dynamic typing. However, in Javascript, the type declaration and variable can be defined anytime.
